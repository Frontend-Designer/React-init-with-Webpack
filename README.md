# React.js with Webpack

## Run

#### Installing

[Node.js](https://nodejs.org/) is Required

Install with NPM

```bash
$ npm install
```

Install with Yarn

```bash
$ yarn
# or
$ yarn install
```

#### Get started

Start to [Webpack Dev Server](https://github.com/webpack/webpack-dev-server)

```bash
$ npm run dev
```

Build with the distribution(production) version.

```bash
$ npm run prod
```

## Available Vendors

- React.js
- EJS
- SCSS
